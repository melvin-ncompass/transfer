# Space Landing Page (Beyond)

A high-end, minimalist landing page for **Beyond**, built using the **"Organic Precision"** aesthetic standards. This project serves as a showcase for sustainable craftsmanship and technical excellence in space exploration.

## Project Overview

- **Core Aesthetic:** "Organic Precision" (inspired by Floema) — a balance of industrial precision and organic elements.
- **Main Technologies:**
    - **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
    - **Styling:** [Tailwind CSS](https://tailwindcss.com/)
    - **Animations:** [Framer Motion](https://www.framer.com/motion/)
    - **Icons:** [Lucide React](https://lucide.dev/)
- **Key Resources:**
    - `pro-frontend/`: Contains the local definition of the `pro-frontend` skill, including design principles and technical specs.

## Architectural Structure

- `app/`: Application routes and layouts.
    - `page.jsx`: The main landing page, implementing the hero, mission, and fleet sections.
    - `globals.css`: Global styles and Tailwind directives.
- `components/ui/`: High-end UI components.

## Development Conventions

### 1. Organic Precision Principles
- **Aesthetic Contrast:** Balance sharp wireframes with soft shadows and organic textures.
- **Generous Whitespace:** Maintain at least `120px` (`py-section`) between major sections.
- **Minimalism:** Remove non-essential UI. Let high-quality typography and imagery lead.

### 2. Typography & Styling
- **Headings:** Use `font-bold`, `tracking-tighter`, and `uppercase` for an authoritative look.
- **Body:** Use `leading-relaxed` for legibility.
- **Colors:** High-contrast monochrome base (Black/White) with subtle muted accents.
- **Spacing:** Adhere to an 8px grid. Use the custom `spacing: { 'section': '120px' }` for vertical padding.

### 3. Animation Patterns
- **Reveal Effects:** Use `initial={{ opacity: 0, y: 20 }}` with a custom ease for entry animations.
- **Interactive Micro-interactions:** Prefer subtle scaling or smooth color shifts on hover (e.g., using `framer-motion`).
- **Performance:** Use `priority` on the Hero image and keep animations lightweight.

### 4. Component Patterns
- **Self-Containment:** Components should be modular and accept an optional `className` for layout adjustments.
- **Type Safety:** Prefer TypeScript (`.tsx`) for components in `components/ui/`.
