# Pro Frontend (Floema Standards)

## Overview
This skill enables the creation of "Organic Precision" frontends: a minimalist, high-contrast aesthetic that balances technical precision with natural elements. It prioritizes editorial typography, generous whitespace, and smooth, immersive interactions.

## Guidelines
When building a frontend with this skill, adhere to the following core principles:

1. **Aesthetic**: Use the [Design Principles](references/design-principles.md) to guide colors, typography, and spacing.
2. **Tech Stack**: Follow the [Tech Specifications](references/tech-specs.md) for Next.js, Tailwind, and Framer Motion patterns.
3. **Hierarchy**: Ensure bold headings and breathable layouts.

## Usage Scenarios

### 1. Initializing a Project
"Set up a professional landing page for a sustainable architecture firm."
- **Action**: Use the tech stack defined in `tech-specs.md`.
- **Focus**: Configure Tailwind with the recommended font families and spacing tokens.

### 2. Designing a Hero Section
"Design a hero section for a product gallery."
- **Action**: Apply full-bleed imagery and centered, high-impact typography.
- **Animation**: Use Framer Motion for a smooth "reveal" effect.

### 3. Reviewing for Quality
"Audit my current landing page for professional standards."
- **Checklist**:
  - [ ] Are section margins at least 100px?
  - [ ] Is header tracking set to tight/tighter?
  - [ ] Does the color palette use high-contrast or earthy tones?
  - [ ] Are animations smooth and subtle (not abrupt)?
